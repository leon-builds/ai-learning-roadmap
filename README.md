# AI Learning Roadmap

> 系统化 AI 学习路线与决策指南 | 帮助不同人群高效规划学习路径

## 简介

本仓库专注于整理 **AI 学习路线** 和 **方法论框架**，目标是帮助不同背景的学习者更清晰地规划学习过程，而不是直接提供大量资源链接。

在 AI 快速发展的今天，很多人面临的问题不是“没有资源”，而是“不知道从哪里开始”“学了之后怎么应用”“不同阶段应该重点突破什么”。本仓库希望通过结构化的路线和决策框架，降低学习过程中的迷茫感。

**适合以下人群：**
- AI 零基础的新手
- 想用 AI 提升日常办公效率的上班族
- 内容创作者（写作、视频、设计方向）
- 有一定基础、想系统进阶或用于开发/变现的人

## 为什么需要学习路线？

很多人在接触 AI 后容易出现以下情况：
- 同时尝试多个工具，精力分散
- 学了很多 Prompt 技巧，但实际工作中用不上
- 不知道不同阶段该重点掌握什么能力
- 学习过程缺乏系统性，容易半途而废

一个清晰的学习路线可以帮助你：
- 明确当前阶段的目标
- 减少无效尝试
- 建立可衡量的进步感

## 新手入门路线（0 基础推荐）

**目标**：在 5-8 周内建立扎实的 AI 使用基础，能够在日常工作和学习中熟练、有效地使用主流 AI 工具。

### 阶段一：建立基础认知与 Prompt 能力（建议 7-10 天）

这个阶段的核心不是学习具体工具，而是理解 AI 的能力和边界，并掌握 Prompt 编写的基础方法。

**重点内容：**
- AI 是什么、能做什么、不能做什么
- Prompt 的基本结构（角色、任务、格式、约束）
- 常见 Prompt 问题及优化方法
- 简单场景练习（提问、总结、 brainstorm）

**阶段目标：**
- 能够写出结构相对清晰的 Prompt
- 理解不同模型的差异（ChatGPT、Claude、Gemini）
- 建立对 AI 输出的判断能力

**建议做法：**
- 每天练习 3-5 个不同场景的 Prompt
- 把写得好的 Prompt 保存下来，形成个人 Prompt 库
- 多观察 AI 的输出，总结它的优势和局限

### 阶段二：工具熟悉与选择（建议 5-7 天）

这个阶段的重点是了解主流工具的特点，并根据个人需求确定主力工具。

**重点内容：**
- ChatGPT、Claude、Gemini 的核心差异对比
- 不同工具在写作、编程、分析等场景的表现
- 如何根据任务类型选择合适工具
- 工具的进阶功能介绍（自定义 GPT、Projects、Artifacts 等）

**阶段目标：**
- 明确自己目前最适合使用哪个工具
- 掌握主力工具的常用高级功能

**建议做法：**
- 用同一个任务在不同工具上测试，观察差异
- 重点深入学习 1 个工具，而不是平均用力

### 阶段三：场景化应用与能力巩固（建议 2-3 周）

这个阶段要把前面学到的知识应用到真实场景中，形成个人使用习惯。

**重点内容：**
- 办公场景应用（邮件、报告、会议纪要、数据整理）
- 学习场景应用（知识整理、题目解答、论文辅助）
- 内容创作场景应用（写作框架、脚本生成、创意 brainstorm）
- 个人 Prompt 体系搭建

**阶段目标：**
- 能够在日常工作中自然使用 AI
- 形成一套属于自己的 Prompt 使用方法
- 建立持续迭代的习惯

**建议做法：**
- 每天至少用 AI 解决 1-2 个真实问题
- 定期复盘哪些 Prompt 效果好、哪些需要优化
- 把常用 Prompt 整理成个人知识库

### 阶段四：体系化与持续进化（长期）

这个阶段已经不再是“学习 AI”，而是把 AI 变成个人能力的一部分。

**重点方向：**
- 建立个人 AI 使用体系
- 学习更复杂的 Prompt 技巧（Chain of Thought、Few-shot 等）
- 探索 AI 与其他工具的结合
- 根据个人目标拓展进阶方向（编程、Agent、变现等）

## 决策指南

在开始系统学习前，建议先思考以下问题：

- 你目前最想解决的问题是什么？（效率提升、内容生产、编程开发、商业变现等）
- 你每周能投入多少时间进行学习和实践？
- 你更倾向于工具型使用，还是愿意投入时间学习方法论和进阶技巧？

根据不同回答，推荐的学习重点和路径也会有所不同。建议先明确自己的核心目标，再选择对应的路线。

## 如何使用本仓库

1. 先阅读「决策指南」，明确自己的学习方向和目标
2. 选择对应的学习路线，按照阶段逐步推进

4. 学习过程中可以随时返回查看更新内容
